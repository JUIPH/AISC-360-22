# Diseño de elementos de Acero (AISC Specification)

Este proyecto es un librería/herramienta para el diseño de miembros estructurales de acero, siguiendo las especificaciones del **AISC 360-22**

El objetivo es facilitar el cálculo de resistencia de diseño (LRFD) para perfiles estándar laminadas en caliente y armados. 

## Caracteristicas Principales 
* **Verificación de Miembros:**
*  **Tensión:** Verificación por fluencia en la sección bruta y ruptura de la sección neta.
*  **Compresión** Cálculo de carga crítica de pandeo (flexionantes, torsionales y flexo-torsional)
*  **Flexión:** Análisis de momentos plásticos, pandeo lateral-torsional  (LBT)
